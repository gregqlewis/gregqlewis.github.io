### Purple Team Home Lab
**Status**: Ongoing | **Focus**: Offensive & Defensive Security

A comprehensive security testing environment combining attack and defense capabilities:

- **Offensive Tools**: Kali Linux on Raspberry Pi 4 for penetration testing and security assessments
- **Vulnerable Targets**: Metasploitable 2 for practicing exploitation techniques
- **Detection Stack**: Wazuh SIEM with OpenSearch for threat detection and log analysis
- **Network Security**: pfSense firewall with proper network segmentation
- **Additional Tools**: Graylog for centralized logging, MongoDB for data storage, CrowdSec for collaborative security

**Technologies**: Docker, Terraform, AWS, Kali Linux, Wazuh, pfSense
